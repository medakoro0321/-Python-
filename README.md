## 使い方
ダウンロード >> Pythonで実行 >> 以下の手順をする

このコードを実行した後、X hoge Z hugaのような数値が出ると思います

そしたらXZをそれぞれ16進数で解読した後、FCE0のような下位4桁のみ取り出してください

後はそれを2byte符号付き16進数から10進数に戻すだけです。
